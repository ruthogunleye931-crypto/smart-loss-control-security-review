Project Overview

This repository documents the structured security testing conducted by my team and I on an AI-integrated retail inventory web application.

Our objective was to ensure the backend API was not only functional but secure before final submission.

Scope of Assessment

Port & Service Verification (Nmap)

Endpoint Enumeration (Gobuster)

HTTP Security Header Review

Authentication & Access Control Testing

CORS Configuration Review

HTTPS Considerations

NDPR Compliance Alignment

Encryption Review (AES-256)

Key Outcomes

No critical vulnerabilities identified

Authentication properly enforced (401 validation confirmed)

Security headers correctly configured

Compliance considerations integrated

Minor improvements recommended for production
